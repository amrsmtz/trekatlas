# TrekAtlas

A simple app to practice your world knowledge, built with [Gatsby](https://www.gatsbyjs.com).

Live at [trekatlas.netlify.app](https://trekatlas.netlify.app/).

## Games

- **Guess the Capital** — endless practice mode
- **Guess the Capital with score** — 10 rounds, get your score

## Development

```shell
npm install
npm run develop   # dev server at http://localhost:8000
npm run build     # production build
npm run serve     # serve the production build
```
